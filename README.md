# 🍅 Stoic Pomodoro Timer (CLI)

A simple, focus-oriented Command Line Interface timer built with Python.
Designed to help maintain deep work sessions using the Pomodoro technique.

## 🚀 Features

- 25-minute Work / 5-minute Break cycles.
- Clean terminal UI with auto-clearing.
- Graceful exit handling (Ctrl+C).
- Stoic-inspired messaging for mindfulness.

## 🛠️ How to Run

Ensure you have Python 3.13+ installed.

```bash
python main.py
```
